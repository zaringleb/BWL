available commands:
/help - send this massage
/stat - get your words statistics
/keyboard - send me keyboard
/start - ONLY if you want to restart everything

To begin send a file with words you want to learn separated by '\n'. For now only ASCII symbols are available.

You can reply to a word with:
{yes} (you remember this word)
{no} (you don`t)
{next} (for sending another word)
{delete} (to get rid of this word)

Also you can reply using keyboard.

For other questions, please contact me on github @zaringleb
